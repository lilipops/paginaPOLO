LAN

LAN (Red de Área Local): Una LAN es una red de computadoras limitada a un área geográfica relativamente pequeña, como una oficina o un hogar, que permite la comunicación de dispositivos en esa área.

WAN

Una WAN es una red que abarca una gran área geográfica, como ciudades, países o incluso continentes, y se utiliza para conectar redes locales separadas entre sí a través de tecnologías de comunicación a larga distancia.

TCP/IP

TCP/IP es un conjunto de protocolos de comunicación utilizados en internet y redes, que permite la transmisión de datos de manera confiable y eficiente entre dispositivos conectados en una red.

Servidor

Un servidor es una computadora o dispositivo especializado que proporciona servicios, recursos o información a otras computadoras o dispositivos, conocidos como clientes, a través de una red.

Servidor WEB

Un servidor web es un tipo de servidor que almacena y entrega páginas web y contenido multimedia a los navegadores web de los usuarios cuando solicitan información a través de internet.

Servidor de correo

Un servidor de correo es un servidor que gestiona el intercambio de correos electrónicos, almacenando, enviando y recibiendo mensajes de correo electrónico entre usuarios en una red.

Servidor FTP

Un servidor FTP es un servidor que utiliza el protocolo FTP para permitir la transferencia de archivos entre dispositivos a través de una red, generalmente utilizado para compartir archivos de manera eficiente.

Entorno LAMP

LAMP es un acrónimo que se refiere a un conjunto de software que incluye Linux (sistema operativo), Apache (servidor web), MySQL (sistema de gestión de bases de datos) y PHP (lenguaje de programación). Se utiliza comúnmente para crear aplicaciones web y sitios web dinámicos.

CMS

Un CMS es una aplicación o plataforma que permite a los usuarios crear, gestionar y publicar contenido en línea de manera sencilla, facilitando la administración de sitios web y blogs sin requerir conocimientos técnicos avanzados.
